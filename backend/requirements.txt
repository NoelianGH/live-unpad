# Backend Requirements

## System Requirements
- Node.js >= 18.x
- npm >= 9.x
- PostgreSQL database (or Neon Postgres)
- Prisma CLI (for database migrations and seeding)

## Primary Dependencies
- @langchain/community >= 1.1.29
- @langchain/core >= 1.1.49
- @langchain/groq >= 1.2.1
- @langchain/mongodb >= 0.0.1
- @langchain/ollama >= 0.2.0
- @langchain/openai >= 1.4.7
- @prisma/client >= 6.19.3
- bcryptjs >= 3.0.3
- cors >= 2.8.6
- dotenv >= 16.4.5
- express >= 4.21.0
- jsonwebtoken >= 9.0.2
- multer >= 1.4.5-lts.1

## Development Dependencies
- nodemon >= 3.1.4
- prisma >= 6.19.3
